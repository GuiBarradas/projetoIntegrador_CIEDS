Criando Banco e Formulário:


Primeiro Passo:

Criar um banco de dados no SQL Server e fazer uma tabela com os seguintes campos:

- Código(PK - "Chave Primária").
- Nome (varchar).
- Ativo (int).
- CPF (varchar).
- Data de nascimento (varchar ou date).

(Os campos não devem possuir acentos ou espaçamentos no SQL Server e nem nas variáveis usadas no sistema).

Segundo Passo:

Criar uma tela formulario com os mesmos campos criados no SQL.

Obs: CPF e data de nascimento devem ter suas respectivas máscaras de campos para facilitar o preenchimento e evitar erros.
Template para criação de tabelas nos padrões da empresa disponível no Discord, entre em contato com seu treinador.
